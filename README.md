## How to use

1. **[Right click this link, and download `AD2 Standard.pitchlist`.](https://github.com/nekonenene/AD2-Map-for-StudioOne/blob/master/AD2%20Standard.pitchlist)**
2. Drag & Drop to the keymap of Studio One's piano roll.
3. Save the preset.
